GCC Optimizations exploration
==================================================================
Prequeries
==========
Docker installed 
Optional: cAdvisor, InfluxDB images
Run
==========
```sh
cd / && sudo -s
git clone https://github.com/mboussaa/DECO-GCC.git
cp -r DECO-GCC /shared
cd /shared
chmod +x processing_driver.sh
./processing_driver.sh
```
